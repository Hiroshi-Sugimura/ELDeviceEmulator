# ELDeviceEmulator

# install

まずはElectron-builderを入れておこう。
```
npm i -g electron-builder
```


- for mac

```
cd core
npm i
```

- for windows

```
cd core
npm i
```

## windows

# execute

- for mac /w Electron

```
npm run mac
```


- for win /w Electron

```
npm run win
```

- for only PicoGW

```
npm run pico
```

# Packaging (Electron)

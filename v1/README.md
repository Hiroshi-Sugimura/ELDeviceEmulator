# ELDeviceEmulator

# install

- for mac

```
cd core
npm i
```

- for windows

```
cd core
npm i
```

## windows

# execute

- for mac /w Electron

```
npm run mac
```


- for win /w Electron

```
npm run win
```

- for only PicoGW

```
npm run pico
```

# Packaging (Electron)

```
npm run build-win
```

```
npm run build-mac
```
